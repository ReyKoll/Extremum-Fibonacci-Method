# Поиск экстремума функции методом оптимизации Фибоначчи

**Приложение строит график одной из 4 функций:**

+ y = Sin(x)
+ y = Cos(x)
+ y = x * x - 4
+ y = x - 2

Поиск чисел Фибоначчи (F#) :
```F#
module Fibonacci =
    let rec Get n =
        if n < 2 then 
            1
        else 
            Get (n - 1) + Get (n - 2)
```
Далее эти числа используются для сужения интервала и расчета значения функции.
